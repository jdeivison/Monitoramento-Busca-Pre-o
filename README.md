# 📊 Monitoramento Busca Preço

Este projeto é um dashboard moderno e responsivo para monitoramento de IPs e status de busca de preços em tempo real. Desenvolvido para facilitar a visualização técnica e o acompanhamento de processos do setor.

## 🚀 Funcionalidades
* **Interface Dark Mode:** Design moderno focado em reduzir o cansaço visual.
* **Alternador de Tema:** Suporte a Modo Claro (Light Mode) para diferentes ambientes.
* **Lista de IPs:** Acompanhamento centralizado dos endereços monitorados.
* **Design Responsivo:** Funciona perfeitamente em computadores e dispositivos móveis.

## 🛠️ Tecnologias Utilizadas
* **HTML5** (Estrutura)
* **CSS3** (Estilização e Variáveis de Ambiente)
* **Font Awesome** (Ícones profissionais)
* **Google Fonts** (Fonte Inter)

## 📁 Arquivos do Projeto
* `index.html`: Código principal do sistema.
* `IPS busca Preços.txt`: Lista de referência de IPs configurados.
